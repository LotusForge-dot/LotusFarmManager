# 2. システム構成

Version 5.0

---

## 2.1 システム概要

Lotus Farm Manager は、HTML・CSS・JavaScript のみで構成されたクライアントサイドWebアプリケーションである。

サーバーは使用せず、データはブラウザの LocalStorage に保存する。

そのため、インターネット接続がない環境でも利用可能である。

---

## 2.2 システム構成図

```
┌──────────────┐
│  index.html  │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│    app.js    │
│ 画面・イベント │
└──────┬───────┘
       │
 ┌─────┴─────────────┐
 ▼                   ▼
master.js       storage.js
マスタ管理        保存・読込
       │
       ▼
 LocalStorage
```

---

## 2.3 構成概要

システムは役割ごとにソースコードを分割している。

画面表示やイベント処理は app.js が担当し、各種マスタ管理は master.js、データ保存は storage.js が担当する。

これにより、各ファイルの責務を明確にし、保守性を向上させている。

---

## 2.4 データ保存方式

すべてのデータは LocalStorage に保存する。

保存対象には以下が含まれる。

- 作業記録
- 田んぼマスタ
- 作業マスタ
- 資材マスタ
- 施肥設計

データは JSON 形式で保存・読込を行う。

---

## 2.5 画面構成

現在実装している画面は以下のとおり。

- 作業記録
- 作業履歴
- 設定
- 田んぼマスタ
- 作業マスタ
- 資材マスタ
- 施肥設計

各画面は app.js から切り替えて表示する。

---

## 2.6 モジュール構成

各 JavaScript ファイルは以下の責務を持つ。

| モジュール | 役割 |
|------------|------|
| app.js | 画面表示・イベント処理・画面遷移 |
| master.js | マスタ管理 |
| storage.js | データ保存・読込 |
