# 4. データ構造

Version 5.0

---

## 4.1 概要

Lotus Farm Manager では、各種データを JavaScript オブジェクトとして管理し、LocalStorage に JSON 形式で保存する。

各データ構造は、機能ごとに独立して管理される。

---

## 4.2 作業記録（records）

作業記録は、日々の営農作業を記録するデータである。

| 項目 | 内容 |
|------|------|
| date | 作業日 |
| field | 田んぼ名 |
| work | 作業名 |
| material | 使用資材 |
| amount | 使用量 |
| memo | メモ |

---

## 4.3 田んぼマスタ（fieldMaster）

田んぼ情報を管理する。

| 項目 | 内容 |
|------|------|
| name | 田んぼ名 |
| area | 面積（a） |

---

## 4.4 作業マスタ（workMaster）

作業名称を管理する。

| 項目 | 内容 |
|------|------|
| name | 作業名 |

---

## 4.5 資材マスタ（materialMaster）

資材情報を管理する。

| 項目 | 内容 |
|------|------|
| name | 資材名 |
| unit | 単位 |
| price | 単価 |
| n | 窒素含有率 |
| p | リン酸含有率 |
| k | カリ含有率 |
| works | 使用可能な作業一覧 |

---

## 4.6 施肥設計（fertilizerPlan）

施肥設計は、圃場ごとの施肥計画を管理する。

| 項目 | 内容 |
|------|------|
| year | 対象年 |
| field | 対象田んぼ |
| materials | 資材一覧 |

---

## 4.7 施肥資材（planMaterials）

施肥設計内で使用する資材情報。

| 項目 | 内容 |
|------|------|
| work | 作業 |
| material | 資材名 |
| amount | 使用量 |

---

## 4.8 保存方式

すべてのデータは LocalStorage に JSON 形式で保存する。

保存・読込処理は storage.js が担当する。
