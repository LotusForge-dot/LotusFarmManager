# 5. データ関係図

Version 5.0

---

## 5.1 概要

Lotus Farm Manager では、各データは独立して管理される。

共通して使用する情報（田んぼ・作業・資材）はマスタデータを参照する構成とする。

---

## 5.2 作業記録

```
records
│
├─ date
├─ field ─────▶ fieldMaster
├─ work ──────▶ workMaster
├─ material ──▶ materialMaster
├─ amount
└─ memo
```

---

## 5.3 施肥設計

```
fertilizerPlan
│
├─ year
├─ field ─────▶ fieldMaster
└─ materials[]
      │
      ├─ work ─────▶ workMaster
      ├─ material ─▶ materialMaster
      └─ amount
```

---

## 5.4 マスタデータ

```
fieldMaster
▲
├─ records
└─ fertilizerPlan


workMaster
▲
├─ records
└─ planMaterials


materialMaster
▲
├─ records
└─ planMaterials
```

---

## 5.5 データ参照関係

| データ | 参照先 |
|---------|--------|
| records | fieldMaster / workMaster / materialMaster |
| fertilizerPlan | fieldMaster |
| planMaterials | workMaster / materialMaster |

---

## 5.6 設計方針

各データは必要最小限の情報のみを保持し、共通情報はマスタデータを参照する。

この構成により、以下を実現する。

- データの重複防止
- マスタ更新時の保守性向上
- データ構造の簡素化
- 将来的な機能追加への対応
